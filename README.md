# Mojolicious::Plugin::Iconify

## Usage

```.pl
# Mojolicious
$self->plugin('Iconify');

# Mojolicious::Lite
plugin 'Iconify';
```

```.html
@@ template.html.ep

<html>
<head>
    <%= iconify_js %>
</head>
<body>
    <h1>
        Mojolicious::Plugin::Iconify
    <h1>
    <p>
        Made with <%= icon 'mdi:heart', style => 'color:red' %> by <em>Giterlizzi</em>
    </p>
</body>
</html>
```

## Installation

To install this module type the following:

    perl Makefile.PL
    make
    make test
    make install
